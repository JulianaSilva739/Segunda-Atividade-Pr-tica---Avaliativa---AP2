# Segunda-Atividade-Pr-tica---Avaliativa---AP2
DESENVOLVIMENTO DE APLICAÇÕES ORIENTADA A COMPONENTES

    Tarefa: Criando e Utilizando Componentes em React

Descrição: Nesta tarefa, você irá praticar a criação de componentes e sua utilização em uma aplicação React simples. A aplicação consistirá em um componente principal e um ou mais componentes secundários que serão utilizados dentro do componente principal.

Passos:

Crie uma nova aplicação React utilizando Create React App ou outra ferramenta de sua preferência.
Dentro da estrutura da aplicação, crie um novo componente.
No novo componente, adicione algum conteúdo ou elementos JSX simples, como um parágrafo ou uma lista, para demonstrar seu funcionamento.
Importe e utilize o novo componente dentro do componente principal da sua aplicação, que geralmente é o App.js.
Verifique se a sua aplicação está funcionando corretamente, exibindo o conteúdo do novo componente quando a página é carregada.
Como entregar:

Após desenvolver a aplicação, você deve postar:
Print de tela do resultado obtido
Print do componente principal da sua aplicação, mostrando a chamada para o novo componente
Compartilhar a pasta com os arquivos da aplicação através do Google Drive (para o email caroline.rosa@ulbra.br) ou compartilhar o link do GitHub com os arquivos da aplicação. O repositório no GitHub precisa ser público para que eu possa visualizar os arquivos.
Observações:

Esta é uma tarefa simples para praticar a criação e utilização de componentes em React. Sinta-se à vontade para adicionar estilos CSS ou expandir a funcionalidade da aplicação conforme desejar.
